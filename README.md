## How I work

**Make it work, then make sure it stays safe.** Every project above carries
that second half: the AI agent's tools are read-only and its API client
never writes; TLS verification is on by default; the network design closes
VLAN-hopping vectors and black-holes unused ports. Security isn't a
separate phase — it's a design decision I make while building.

I build with AI in the loop — agents, not just autocomplete. But the rule
is simple: **I don't ship code I can't explain.** The Zammad tools above
ran against a real production deployment; when something broke at 9am,
the person debugging it was me.

Currently going deeper on Linux operations and Docker, closing the gap
between "it works" and "I know exactly why it works."
